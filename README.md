# RandomWorkAlarm_2026-06
作業効率促進用アラームアプリ

1．概要
  指定した時間範囲内でランダムな時間を取得。その時間にアラームが発火し通知を飛ばす。
        通知をタップするとアプリに遷移しタイマーが開始する

2.実装済み
  時間設定
  時間設定後再度設定ボタンを押せないように制御
  設定した時間内からランダムな時間を取得
  取得した時間に通知を飛ばす
  通知をタップするとアプリに遷移しタイマーが開始
  

3.実装予定
  時間設定の欄を手入力からドラムロール式に変更
  通知タップ後、タイマー画面を固定
  通知ではなくアラーム音がなるようにする
